# 🗺️ SU Second-Hand Shop Finder (เว็บระบบค้นหาร้านเสื้อผ้ามือสอง ม.ศิลปากร)

A Web Application project designed for finding second-hand clothing shops around Silpakorn University (Sanam Chandra Palace Campus). This project was built as part of the **Web Application** course with the primary goal of learning how to integrate and work with the **Google Maps API**.

> *"This website might look trash, but we believe it is our proud first step toward becoming good programmers in our career paths."* 🚀

---

## 👥 Our Team & Roles
* **My Friend (The Architect):** Created and designed the core structures, databases, and structural coding layout.
* **Me (The Feature Builder & UI/UX Debugger):** Developed features and solved UI/UX bugs to ensure smooth operation.

---

## ✨ Features & Learning Objectives
* **Google Maps API Integration:** The main core of this project is learning how to connect with the Google Maps API to place interactive custom markers and display shop coordinates smoothly.
* **Basic Filtering System:** Allows users to filter shops by clothing types and price ranges.
* **Shop Status Display:** Calculates and shows if a shop is currently open or closed based on operating hours.
* **Three-Language Options:** Supports simple text switching for Thai (TH), English (EN), and Chinese (ZH).
* **User Location Pinning:** A feature that lets users dynamically add/pin a new shop directly onto the map.
* **Refactored Code Structure:** The code has been cleaned up and separated into modular files (HTML, CSS, JS, and JSON) for better organization and easier debugging.

---

## 🛠️ Built With
* **HTML5** - Page structure and controls.
* **CSS3** - Layout designs, mobile responsiveness, and sidebar animations.
* **JavaScript (ES6)** - Client-side operations and **Google Maps API** implementation.
* **JSON** - A simple text-based data store for shop information and translations.
